# Run in dev
1. Clone repo
2. Execute
  ```
  yarn install
  ```
3. Install Nst CLI
  ```
  npm i -g @nestjs/cli
  ````
4. Setup DB
```
  docker-compose up -d
```
5. Seed data base
  ```
  http://localhost:3000/api/v1/seed
  ```


## Stack
* MongoDB
* Nestjs
